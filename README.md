# Home & Heart – Eiropas Ģimeņu festivāls

Vienkārša statiska festivāla mājaslapa latviešu valodā.

## Faili

- `index.html` – lapas saturs
- `styles.css` – vizuālais noformējums
- `script.js` – mobilā izvēlne un laika atskaite

## Pievienots no Family Activity Hub

Lapa izmanto Home & Heart logotipu un ES līdzfinansējuma logo no esošās projekta mājaslapas:

- `https://familyactivityhub.com/visuals/home-heart-logo.png`
- `https://familyactivityhub.com/visuals/eu-co-funded.png`

Ja vēlaties, šos attēlus vēlāk var lejupielādēt un ievietot lokālā `visuals/` mapē, lai lapa nebūtu atkarīga no ārējām saitēm.

## Ko vēl nomainīt pirms publicēšanas

- gala programma un precīzi laiki;
- pieteikšanās forma vai reģistrācijas saite;
- festivāla karte;
- jebkādi papildu partneru/koordinatoru logo, ja nepieciešams.
